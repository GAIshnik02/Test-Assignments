# Тестовое задание для Лиги Цифровой Экономики

## Задача про календарь

### Вопрос

Сколько всего уникальных календарей существует вне зависимости от номера года?

---

### Ответ:
 **14 (7 для обычных годов + 7 для високосных)**

 Объяснение: Календарь определяется двумя факторами:
 1. Високосный год или нет (2 варианта)
 2. День недели, с которого начинается год (7 вариантов)
Итого: 2 * 7 = 14 уникальных календарей
